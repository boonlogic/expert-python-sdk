# **save_snapshot()**
<br/>

#### Save a serialized version of the nano
##### input
>`nano_instance_id`    
>>*instance ID of the nano to save*  
>
>`filename`  
>>*name of the file to save the nano to*

##### output
>*`True` if the file was created successfully*

or
>*prints the error code and message and returns `False`*
